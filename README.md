# adhd-tcn-replication

## Data

This project uses the ADHD-200 dataset (Athena-preprocessed release).
Due to size constraints, the dataset is not included in this repository.

### Download

The data can be downloaded from:
- https://www.nitrc.org/frs/?group_id=383

Details about the Athena preprocessing pipeline are available at:
- https://www.nitrc.org/plugins/mwiki/index.php/neurobureau:AthenaPipeline

### Expected directory structure

After downloading and extracting the data, the repository should have the
following structure:

data/
├── raw/
│   ├── aal_tcs/
│   └── phenotypic/
├── processed/
